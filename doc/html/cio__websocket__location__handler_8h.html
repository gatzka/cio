Initializes a websocket handler.

Parameters
handlerThe handler to initialize.
subprotocolsAn array of strings containing the supported subprotocols. Please note that the functions will not copy this array, this array must be available as long as this cio_websocket_location_handler exists!
num_subprotocolsThe number of entries subprotocols contains.
on_connectFunction that will be called if websocket is connected.
location_handler_freeThis function will be called if the client connection for this handler is closed. After this function is called, the memory handler points to will no longer be accessed. Could be NULL.
Returns
CIO_SUCCESS if no error occured.
